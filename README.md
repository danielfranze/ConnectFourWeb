ConnectFourWeb
=====================================
ConnectFourWeb is a web implementation of the game Connect Four.

See also: https://en.wikipedia.org/wiki/Connect_Four.

Installation
--------------------

1. Clone this Repository
2. Open it with IntelliJ IDEA (Scala plugin needed!)
